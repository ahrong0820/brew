# Source 체크리스트

- [ ] 제목과 작성자를 확인했다.
- [ ] DOI, 영상 ID 또는 문서 식별자가 있다.
- [ ] 원본 URL과 접근일이 있다.
- [ ] 자료 유형과 공개 상태를 확인했다.
- [ ] 제조사 자료는 제품 모델과 문서 버전을 기록했다.

## 전문가 영상 추가 확인

- [ ] 전문가 본인 또는 소속 조직의 공식 채널인지 확인했다.
- [ ] 채널명, 채널 URL, 소유 관계와 검증 방법을 기록했다.
- [ ] 재업로드·반응 영상·제3자 편집본인지 확인했다.
- [ ] Observation에는 타임스탬프, 챕터, 화면 정보 또는 공식 설명란 위치가 있다.
- [ ] 자동 자막의 숫자·단위·고유명사를 다른 직접 정보와 대조했다.
- [ ] 후속 영상이 기존 레시피를 대체하는지 확인했다.
