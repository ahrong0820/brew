# 공식 전문가 영상 근거 정책

## 목적

전문가 본인의 공식 영상은 전문가가 직접 공개한 1차 자료로 취급한다. 자막이 없다는 이유만으로 Source 또는 Observation 등록을 거부하지 않는다. 다만 영상의 존재 확인과 영상 속 주장 확인은 분리한다.

## 공식 출처 판정

다음 중 하나를 확인하면 공식 영상 출처로 분류할 수 있다.

- 전문가 본인이 운영하는 공식 채널
- 전문가의 공식 웹사이트나 공식 프로필이 연결한 채널
- 전문가가 소속된 조직이 운영하며 해당 전문가의 발표를 원본으로 게시한 공식 채널

재업로드, 반응 영상, 요약 영상, 제3자 편집본은 전문가 본인의 직접 출처로 분류하지 않는다. 원본을 찾지 못한 경우 Source 후보로 보관할 수 있지만 `ownership`을 `third-party` 또는 `unknown`으로 기록하고 직접 근거처럼 사용하지 않는다.

## Source 등록 기준

공식 채널 영상은 다음 정보를 확인한 뒤 `EvidenceSource`로 등록한다.

- 영상 제목
- 발표자 또는 제작자
- 게시 채널
- 게시일
- 원본 영상 URL
- YouTube 영상 ID
- 채널 소유 관계와 검증 방법
- 접근일

공식 채널 여부가 확인되면 자막 유무와 관계없이 Source 등록이 가능하다.

## Observation 등록 기준

Observation은 영상에서 실제로 확인한 주장만 등록한다. 다음 위치 정보 중 하나 이상을 남긴다.

- 발언 시작 타임스탬프와 필요 시 종료 타임스탬프
- 영상 챕터
- 화면에 표시된 레시피 카드나 수치의 시점
- 제작자가 작성한 영상 설명란의 문단

자막, 자동 자막, 설명란, 챕터, 화면 정보는 모두 확인 보조 수단이다. 자동 자막만 사용한 경우 숫자, 단위, 고유명사는 영상 화면이나 다른 직접 정보와 대조한다.

## 등록 가능한 내용

- 명시적인 레시피 수치
- 변수 조절 방향과 조건
- 분쇄도, 온도, 비율, 유속, 교반, 추출 시간에 관한 지침
- 특정 원두, 배전도, 가공 방식, 드리퍼 또는 필터에 대한 적용 조건
- 이전 레시피를 수정하거나 대체한다는 직접 설명

영상 제목이나 썸네일만으로 추론한 내용, 제3자의 요약, 맥락을 확인할 수 없는 짧은 클립은 Observation으로 등록하지 않는다.

## 개정과 계보

동일 전문가가 후속 영상에서 레시피를 변경한 경우 새 Observation을 추가하고 기존 Observation을 삭제하지 않는다. 기존 자료는 `superseded` 관계 또는 제한 근거로 보존하여 어느 시점의 레시피인지 추적할 수 있게 한다.

## 추천 규칙 연결

공식 영상이라는 사실만으로 활성 추천 규칙에 연결하지 않는다. Observation 검수, 적용 조건 일치도, 독립 근거 수, 제한·반박 근거를 평가한 뒤 CandidateRule에 연결한다. 전문가 영상 수집 PR과 추천 계산 변경 PR은 분리한다.
