# 산지 조건 일치 모델

## 목적

산지 정보를 국가별 고정 향미 규칙으로 사용하지 않고, 근거 Observation의 적용 조건과 대상 원두의 조건이 얼마나 가까운지 평가한다. 이 점수는 근거의 관련성을 계산할 뿐 활성 추천값을 직접 변경하지 않는다.

## 판정 순서

1. 동일 국가가 명시되면 `match`
2. 국가가 다르거나 생략됐지만 동일 산지 그룹이면 `partial`
3. 국가와 산지 그룹이 모두 다르면 `mismatch`
4. 대상 원두에는 산지가 있지만 근거에 산지가 없으면 `unknown`
5. 대상 조건에서 산지를 평가하지 않으면 `not-applicable`

더 구체적인 국가 일치가 산지 그룹보다 우선한다. 동일 산지 그룹은 국가 일치와 동일한 근거로 취급하지 않는다.

## 다른 원두 조건과의 관계

산지 판정은 다음 조건을 대체하지 않는다.

- 가공 방식
- 배전도
- 품종
- 고도
- 로스팅 후 경과일

예를 들어 에티오피아 워시드와 에티오피아 내추럴은 산지 차원에서는 일치할 수 있지만 가공 방식 차원에서는 별도로 평가된다.

## 가중치 정책

산지 차원은 전체 조건 일치 점수의 0.11을 차지한다. 모든 조건 차원의 가중치 합은 1로 유지한다. 알려진 부분 일치는 정보가 없는 상태보다 높은 관련성을 갖도록 `partial > unknown > mismatch` 순서로 점수를 부여한다.

## 제한

현재 `OriginCountry`와 `OriginGroup`은 비교할 수 있지만 세부 지역 필드는 아직 없다. 국가 내부 지역, 농장, 품종과 고도까지 포함한 확장은 별도 데이터 모델 변경으로 처리한다.
