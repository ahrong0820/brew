# 그라인더 설정-대표 입도 참조 곡선

## 목적

사용자가 목표 대표 입도(μm)를 입력하면 K-Ultra 다이얼 또는 Baratza Encore 기본형 클릭의 시작점과 탐색 범위를 제안한다.

대표 입도는 실제 분쇄물의 모든 입자가 같은 크기라는 의미가 아니다. 입도 분포를 하나의 숫자로 단순화한 참고값이며, 최종 설정은 추출 유속과 맛으로 조정한다.

## 적용 기준

### 1Zpresso K-Ultra

- 사용자 기준: 버가 스치지 않기 시작하는 최초 지점을 0으로 맞춘 버 비접촉 영점
- 제조사 사양: 클릭당 20μm 조절 정밀도
- 주의: 제조사의 20μm는 조절 기구의 이동 간격이며 분쇄 입자의 대표 크기와 동일하지 않다.
- 초기 곡선은 제조사 조절 구조, 공개된 추출 방식별 설정 범위와 일반적인 입도 구간을 종합한 보수적인 참고값이다.

### Baratza Encore

- 대상: Encore ESP가 아닌 기본형 Encore
- 기준: 출고 상태, 기본 버, 내부 보정링 미변경
- 제조사 사양: 총 40단계 설정
- 초기 곡선은 공개된 추출 방식별 설정 범위와 일반적인 입도 구간을 종합한다.
- 버 마모, 조립 상태와 내부 보정링 위치에 따라 같은 클릭에서도 결과가 달라질 수 있다.

## 구현 원칙

1. 단일 설정을 정답처럼 제시하지 않고 시작점과 탐색 범위를 함께 제공한다.
2. 곡선 사이 값은 선형 구간 보간으로 계산한다.
3. K-Ultra는 기본 ±80μm, Encore는 기본 ±120μm 범위를 설정값 범위로 변환한다.
4. 참조 범위를 벗어난 목표값은 가장 가까운 끝점으로 제한한다.
5. 향후 실제 측정값과 성공 추출 기록이 쌓이면 기본 곡선보다 개인 기록을 우선한다.
