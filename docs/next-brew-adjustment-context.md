# Next-brew adjustment context

The adjustment drawer keeps the existing one-variable diagnostic decision and adds presentation context from the same recipe-scoped brew profile.

Displayed context:

- source recipe name
- actual and target brew time
- recorded tasting result
- personalization stage: trial, provisional, or stable
- total and successful brew counts for the same profile
- variables that must remain unchanged during the next comparison brew

A profile becomes provisional after one successful brew and stable after two or more successful brews. Different barista recipes remain isolated because the profile identity already includes the selected source recipe ID.
